Angular Guacamole Client

Environment:

  1. guacamole/guacd. Docker image is a better way for adding it to a developing environment
    • Install Docker
    • Pull guacd image: docker pull guacamole/guacd
    • Running guacd: docker run --name some-guacd -d -p 4822:4822 guacamole/guacd
  2. web-sockets⇄guacd proxy - guacamole-test-server
    • Download guacamole-test-server
    • Download and install Maven
    • Download and install JDK
    • Run: mvn package
    • Run: java -jar target/gts.jar --guacd-host 1.2.3.4 --guacd-port 4822 --port 8080

      1.2.3.4 is ip address of your guacamole/guacd, probably it will be an ip address of your local network interface e.g. 192.168.1.55

  3. I had second computer with windows 7 in my local network. I used it as rdp server.

    Edit src\app\shared\component\remote-desktop\remote-desktop.component.ts with right parameters of ip address: line 26

  4. Run npm install
  5. Run npm run start
